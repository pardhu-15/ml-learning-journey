
Problem Statement:
Spam Message Classifier Using Machine Learning

Dataset Used:
SMS Spam Collection Dataset(taken from Kaggle)
Features of dataset:
1. label
2. text

Approach/Workflow:
1. Collect and Load the dataset
2. Data Analysis (EDA)
3. Data Wrangling (drop extra and unnecessary columns)
4. Train Test Split
5. Feature Extraction using TF-IDF
6. Train the Model using Logistic Regression
7. Testing the Model
8. Saving the models Successfully

Algorithms used to Train the Model:
Logistic Regression

Text Vectorization:
TF-IDF Vectorizer

Evaluation Metrics:
Accuracy
Confusion Matrix
Classification

Final Accuracy:
96.42%
